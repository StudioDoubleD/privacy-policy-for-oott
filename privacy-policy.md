# Privacy Policy - OOTT (Subtitle for Web Video)

**Last updated: 2025-02-09**

## Overview

OOTT is a Chrome extension that allows users to apply custom SRT/VTT subtitle files to web videos. This extension is designed with privacy as a core principle.

## Data Collection

**We do not collect any data.**

- No personal information is collected
- No browsing history is tracked
- No usage analytics are gathered
- No cookies are used for tracking purposes

## Data Storage

All data is stored locally on your device only:

- Subtitle file content is processed entirely within your browser
- Language preferences are stored in your browser's local storage
- No data is transmitted to any external server

## Network Communication

This extension does not communicate with any external servers. All subtitle processing, sync adjustment, and cue browsing are performed locally within your browser.

## Permissions

This extension requires the following permissions solely for subtitle functionality:

- **scripting**: To inject subtitle tracks into video elements on web pages and read video playback time for sync adjustment
- **webNavigation**: To detect page navigation events and automatically reapply subtitles after ads or page transitions
- **sidePanel**: To provide the subtitle control UI in Chrome's side panel
- **host_permissions (all URLs)**: To enable subtitle application on any website, not limited to specific video platforms

## Third-Party Services

This extension does not use any third-party services, APIs, or SDKs.
